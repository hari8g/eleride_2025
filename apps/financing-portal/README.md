# Eleride Financing Portal (Buy-back Underwriting Dashboard)

Financing-company portal to monitor:
- Vehicles leased to fleet partners (operators)
- Partner performance rollups
- Buyback value estimates (heuristic MVP)

## Run

```bash
cd apps/financing-portal
npm install
npm run dev
```

Open `http://localhost:5178`.

## Demo

1) Create tenant (OTP) in the UI
2) Click **Seed demo portfolio**
3) Explore Portfolio → Partners → Vehicles (Buyback)
